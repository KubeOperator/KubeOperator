#  FIT2OPENSHIFT 安装文档
## 环境
-  CPU: 64位双核处理器
-  内存: 4G DDR3
-  操作系统: CentOS 7+
-  Docker: 推荐使用 18.03+


## 安装

**1.1 下载或 Clone 项目**

    $ git clone https://github.com/fit2anything/fit2openshift.git 

**1.2 执行安装脚本**

    $ cd fit2openshift/installer
    $ ./install.sh

**1.3 启动 FIT2OPENSHIFT**

    $ service f2o start
